# Upstream Attribution

ZealRN is an independent application based on [Zeal](https://github.com/zealdocs/zeal), the offline documentation browser. The original Zeal source, authorship, GPL-3.0-or-later licensing, and copyright headers remain part of this repository.

ZealRN adds page-linked Learning Notes, a persistent Start Note, Markdown editing and exports, an offline Web Playground, appearance settings, onboarding, and developer-tool integration.

ZealRN is maintained independently. The upstream Zeal project does not endorse, sponsor, or support ZealRN. Upstream links are retained where they identify original source, licenses, attribution, docset services, or synchronization context.

Contributions to ZealRN are developed in the organization repository at <https://github.com/zealrn/zealrn-desktop>. Changes are synchronized selectively and are not automatically pushed to upstream Zeal.
