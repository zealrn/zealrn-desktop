# ZealRN 0.1.0 Linux Testing

## AppImage

```bash
sha256sum -c dist/SHA256SUMS
./scripts/install-user-local.sh
~/.local/bin/zealrn
```

Verify the Quick Tour, Start Note persistence, docset download and offline reopening, page-linked notes, All Notes search, every export format, Web Playground, embedded terminal prompt, and System/Light/Dark appearance.

Uninstall without removing user data:

```bash
./scripts/uninstall-user-local.sh
```

## Debian Package

Inspect without installation:

```bash
dpkg-deb --info dist/zealrn_0.1.0_amd64.deb
dpkg-deb --contents dist/zealrn_0.1.0_amd64.deb
```

Optional system installation:

```bash
sudo apt install ./dist/zealrn_0.1.0_amd64.deb
```

Removing the package must not remove user notes, settings, exports, or docsets. Original Zeal and its Flatpak data must remain unchanged.
