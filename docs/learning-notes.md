# Learning Notes

**Start Note** is an always-available scratchpad stored under ZealRN's reserved system identity. It is not attached to a documentation URL.

Opening a docset page switches the panel to that page's note. ZealRN identifies page notes by docset plus normalized relative page path, so temporary local-server ports do not affect note identity. Returning to Start Note or another page flushes pending edits first.

Notes use plain Markdown text and autosave after about one second. `Ctrl+S` saves immediately. Edit/Preview, zoom, formatting, find, counts, line wrap, Expand, and Focus change presentation only; they do not change the database format.

**Add Selection** appends selected documentation text as a Markdown blockquote. It never executes the selection or modifies the documentation page.

**All Notes** searches note content, page title, docset, and relative path. One Start Note and one main note per documentation page are supported in 0.1.0.
