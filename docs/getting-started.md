# Getting Started

On first launch, the Quick Tour explains the main workflow. Reopen it later with **Help > Quick Tour**.

1. Open **File > Docset Library** and download the documentation you need.
2. Use **Start Note** immediately, even before opening documentation.
3. Open a docset page to switch Learning Notes to that page's note.
4. Open **View > Show Web Playground** to try HTML, CSS, and JavaScript.
5. Open **View > Show Developer Terminal** to use installed command-line tools.
6. Export notes or a database backup regularly.

The welcome-page checklist records these local actions and can be dismissed or reset in Preferences. ZealRN works without an account and keeps notes on the device.

## Keyboard Shortcuts

- `Ctrl+S`: save the current note
- `Ctrl++`, `Ctrl+-`, `Ctrl+0`: change or reset note zoom
- `Ctrl+F`: find in the focused note or documentation page
- `Ctrl+Backtick`: show Developer Terminal
- `Ctrl+Shift+L`: open Docset Library
- `Ctrl+B`: toggle the documentation sidebar
- `Ctrl+K` or `Ctrl+L`: focus documentation search
- `Ctrl+Shift+7`, `Ctrl+Shift+8`, `Ctrl+Shift+9`: numbered list, bullet list, or blockquote
- `Alt+Left`, `Alt+Right`: documentation back and forward

ZealRN Web is a separate browser trial at <https://abnzrdev.github.io/zealrn-web/>. The desktop application supports the full downloadable docset catalog and native local integration.
